Admin applications¶

This section list the applications and applications mechanics of the site.

List existing applications¶

The existing applications can be listed in /admin/apps/application/, from there the following actions can be performed:

  • View the details of the application.
  • Filtered the applications by: stage, category, status or date.
  • Search the applications by their contents.

Unpublishing an application¶

In case an application needs to be unpublish it can be done from the detail menu of the application, this can be done by changing the status of the application as draft or removed.
